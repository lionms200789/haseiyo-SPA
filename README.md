# 電商網站 - HaSeiYo
> 此SPA專案為虛擬購物商店，銷售主題為男性服飾，並包含前台以及後台。

### 簡介

前台功能 : 

* 商品展示(產品分類、單一產品詳細資訊)
* 購物車及模擬簡易結帳
* 商品收藏清單
* 站內產品搜尋
* 切換瀏覽方式、商品排序

後台功能 : 

* 產品管理(新增、刪除、修改)
* 優惠碼管理
* 訂單列表管理

### 該專案使用了

* Vue.js 
* Vue CLI 
* Vue Router 路由模擬
* Bootstrap 4 
* JavaScript (ES6、LocalStorage 等..)
* 使用SCSS 撰寫與管理樣式
* AJAX串接RESTful APIs
* ESLint Standard 

### 使用的Vue-Plugin

* Vue-axios 
* VeeValidate
* Vue-loading-overlay
